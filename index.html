다운로드 사용 가이드

sc advisor 소개

  • 클릭 한 번으로 PC 보안을 점검하고 자동으로 취약점을 조치해주는 보안 점검 도구
  • 한국인터넷진흥원(KISA)의 보안 기준을 기반으로 만들어져, 누구나 손쉽게 안전한 컴퓨터 환경을 구축
  • 보안 지식이 없어도, 복잡한 설정 없이도, 쉽고 빠르게 보안 점검과 개선이 가능

서비스 대상

  • 중소기업 및 개인 사업자
    • 비용과 인력 부족으로 보안 관리가 어려운 기업을 위한 맞춤형 솔루션
  • 교육기관 및 공공기관
    • 도서관, 학교 등 공용 PC를 다수 운영하는 기관의 기본 보안 점검용
  • 일반 개인 사용자
    • 클릭 몇 번으로 PC 보안을 강화하고 싶은 모든 사용자

서비스 절차

  1. 프로그램 설치 및 실행- 공식 웹사이트에서 sc advisor 다운로드
  2. 보안 점검 시작- '취약점 검사 시작' 버튼 클릭
  3. 결과 확인- 점수 및 위험도 별로 항목 표시 (안전/경고/위험)
  4. 자동 보안 조치 실행- 취약점 항목 클릭 시 자동으로 조치 적용
  5. 보안 완료 및 리포트 제공 [유료 서비스]

[보안 조치 이력 로그화 및 리포트 기능 제공]

  • 기업 사용자를 위한 유료 기능
  • 취약점 점검 결과와 취약점 조치 내용을 기록하고, 리포트 형식으로 출력할 수 있는 기능
